# Get-da

## Description

Get and save DoujinAntena.com images.

## Usage

    Usage: get-da <URL>

    URL: The first page of E-Hentai image page.
         e.g. http://doujinantena.com/page.php?id=c97a6999bc416467f560c62460fb1a88

## Build

This is written in Go Language. So, just build it:

    # cd get-da
    # go build
